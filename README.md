# SemiTNet — سگمنتیشن و شناسایی دندان

> فقط Python 3.10 و `.venv` استفاده می‌شود؛ Conda/Anaconda لازم نیست.

## ۱. نصب پیش‌نیازها

```bash
make install
```

## ۲. دانلود دیتاست و مدل

```bash
make download
```

## ۳. اجرای Smoke Test

```bash
make smoke
```

## ۴. اجرای کامل

```bash
make full
```

> اجرای کامل به Linux و کارت گرافیک NVIDIA/CUDA نیاز دارد.
